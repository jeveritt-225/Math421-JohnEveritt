  1. For two categorical variables: barplot

    df <- read_csv("C:/Users/student/Downloads/2019WH.csv")
df <- df %>%
  mutate(
    Socialsupportgroup = if_else(
      Socialsupport > mean(Socialsupport, na.rm = TRUE),
      "Above Average",
      "Below Average"
    )
  )
df <- df %>%
  mutate(
    Healthylifegroup = if_else(

      "Below Average"

  )
df %>% ggplot()+
  geom_bar(mapping=aes(x=Healthylifegroup, fill=Socialsupportgroup), position = 'fill')

    Countries with above average healthy life scores usually have above average scores in social support, and the same goes for the reverse, showing a strong relationship between the two.
